🎮 Настройка Renode (опционально)
bash
# Для Windows
choco install renode

# Для Linux
echo "deb https://downloads.renode.io/renode-debian testing main" | sudo tee /etc/apt/sources.list.d/renode.list
wget -qO - https://downloads.renode.io/renode-debian/renode.gpg.key | sudo apt-key add -
sudo apt update && sudo apt install renode

🔍 Диагностика проблем
❌ Распространенные проблемы и решения
Проблема: "Port 8888 already in use"
Решение:

bash
# Найди процесс использующий порт
netstat -ano | findstr :8888
# Заверши процесс или измени порт в config.py
Проблема: "ImportError"
Решение:

bash
# Переустанови зависимости
pip install -r requirements.txt
Проблема: Renode не запускается
Решение:

bash
# Используй Python эмулятор вместо Renode
cd stm32_emulator
python stm32_emulator.py
